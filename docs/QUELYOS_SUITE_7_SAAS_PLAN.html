Risques & Mitigation

Duplication code entre apps
Monorepo strict + @quelyos/ui-kit + @quelyos/api-client. Tout composant partagé = package.
Confusion branding (7 marques)
Sous-marques « by Quelyos » + landing page unifiée quelyos.com avec navigation vers chaque SaaS.
Effort développement sous-estimé
MVP ultra-minimal par app. Itérer avec pilotes. Ne jamais lancer 2 SaaS en parallèle sans équipe dédiée.
Cannibalisation ERP complet
Pricing bundles attractifs (-10 à -22%). Upsell naturel SaaS → Full Suite. ERP = offre premium.
Complexité infrastructure
1 backend unique (Odoo 19). CI/CD automatisé par app. Monitoring centralisé (super-admin).
Support client multi-app
Knowledge base commune. Chatbot AI. Quelyos Support utilisé en interne. Documentation unifiée.
